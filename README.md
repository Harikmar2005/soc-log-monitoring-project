# SOC Log Monitoring Project

## Overview
A Security Operations Center (SOC) simulation project focused on monitoring, analyzing, and responding to security events using log analysis.

## Tools Used
- Splunk
- Linux (Ubuntu/Kali)
- Wireshark

## Key Activities
- Log monitoring and alert analysis
- Detection of brute-force login attempts
- Dashboard and alert creation
- Incident response documentation

## Outcome
Improved understanding of SOC workflows, threat detection, and security monitoring.
